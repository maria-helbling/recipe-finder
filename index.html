About the data

Legend

Colors

box - an improving situation box - a stable situation box - a deteriorating situation

Numbers

Active cases - shows per every 100k people in that location, how many currently are considered to have COVID-19

New daily cases - shows per every 100k people in that location, how many new cases were reported yesterday

14 day Trend - for active cases, we take the reported number of active cases in each day for the last 14 days. We calculate the slope of the trend line over time and convert it so it shows the % change relative to the average number of cases in that 14 day period. On a broad level this shows whether the situation is improving, deteriorating or stable. On a more nuanced level, the trend means that if there are currently 100 active cases and the trend is -10%, then IF this trend is maintained, the location would be active case free in 10 days. Of course this is not really how the real workd works and a lot of variables affect case count, but it gives some idea of how the situation is changing

Sources


We use thre APIs to find this data for you. The most important one is the COVID data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University.

This data is for informational purposes only and does not serve in any advisory capacity. Like with any statistical illustration, this shows the best available info snapshot at a given moment. The data at this repository is updated often and retroactively, so the results for each destination are subject to change.
